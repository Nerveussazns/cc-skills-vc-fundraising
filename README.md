## 📋 What Is cc-skills-vc-fundraising?

cc-skills-vc-fundraising is a tool designed to help startup founders plan and improve their fundraising efforts. It uses AI to analyze your pitch and offer feedback. The app also helps you shape fundraising strategies and manage communication with investors. It is built with insights from top venture capital firms like Sequoia, a16z, Benchmark, and YC.
